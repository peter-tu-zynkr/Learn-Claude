## Purpose

A personal learning journal for exploring AI-powered coding tools, modern development workflows, and building with Claude Code + Cursor + GitHub.

## Repository Structure

- `learn-coding-progress.md` — Learning checklist synced from Notion, tracking progress across Claude Code, GitHub, Cursor, MCP, and more
- `Term glossary.md` — Plain-language glossary of coding terms (frontend, backend, cloud, AI, databases, etc.)
- `deeplearning.ai/` — Course notes (e.g., Claude API, Code Execution, Files API)
- `CLAUDE.md` — This file; project-level instructions for Claude Code

Notion, Gmail, Context7, Slack, WhatsApp, Jira/Confluence, Figma, X, Google Calendar, Agent Browser Skills

## Notes

- This is an exploratory/learning environment — prefer clear, readable code over optimized or production-hardened solutions.
- When adding new examples or experiments, organize them in descriptively named files or subdirectories so the purpose is self-evident.
- `learn-coding-progress.md` is the source of truth for learning progress — sync it from the Notion "Learn Coding" page when asked.
- Never commit files matching `*.keys.json` or `*.credentials.json` — these contain secrets.
